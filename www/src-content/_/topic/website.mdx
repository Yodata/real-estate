```json
{
  "topic": "realestate/website#appointmentrequest",
  "data": {
    "type": "AppointmentRequest",
    "agent": {
      "type": "Person",
      "name": "Bob",
      "email": "bob@example.com",
      "telephone": "1+000-867-5309",
      "identifier": {
        "vendornameuserid": "abc123"
      }
    },
    "object": {
      "type": "Comment",
      "id": "http://example.com",
      "name": "string",
      "url": "http://example.com",
      "text": "Winter is coming",
      "author": {
        "type": "Person",
        "email": "user@example.com"
      },
      "about": {
        "type": "RealEstateListing",
        "id": "http://example.com/listing/1"
      }
    },
    "instrument": "https://{app}.example.com/profile/card#me",
    "recipient": "http://{agent}.example.com/profile/card#me"
  },
  "source": "http://{user}.example.com/profile/card#me",
  "instrument": "http://{supercrm}.example.com/profile/card#me",
  "time": "2019-05-21T01:05:11Z",
  "contenttype": "application/json",
  "@context": "http://realestate.yodata.me/context/v1.json"
}
```


| Name | Type | Description |
|:-----| :--- | :---------- |

| data.agent | object | a website user/visitor <br/>RANGE: [Person](/schemas#person) |
| data.object | object | The object upon which the action is carried out  |
| data.instrument | * | thing used by the agent to perform the action, i.e. an application or website that generated the message.  |
| data.recipient | * | the request recipient <br/>RANGE: [Person](/schemas#person),[Organization](/schemas#organization) |
| source | string&lt;uri&gt;! | the event origin, typically a user  |
| instrument | string&lt;uri&gt; | event producer, typically an app or service  |
| time | string&lt;date-time&gt; | when the event was published  |
| contenttype | string | mimetype of the data property.  default=application/json  |
| @context | string&lt;uri&gt; | message context used for transforming message content  |

```json
    },
    "object": {
      "type": "RealEstateListing",
      "id": "http://example.com/12345"

    "name": "string",
    "url": "http://example.com",
    "recipient": {
      "id": "http://user.example.com/profile/card#me"
    },
    "offerPrice": {
      "type": "PriceSpecification",
      "minPrice": 0,
      "maxPrice": 0,
      "price": 0,
      "priceCurrency": "USD"
    }
  },
  "source": "http://{user}.example.com/profile/card#me",
  "instrument": "http://{supercrm}.example.com/profile/card#me",

  "contenttype": "application/json",
  "@context": "http://realestate.yodata.me/context/v1.json"
}
```


| Name | Type | Description |
|:-----| :--- | :---------- |
| topic | string! | realestate/website#propertyoffer  |
| data | object! | an agent makes an offer to buy the object  |
| data.type | string! | OfferAction  |
| data.agent | object | a website user/visitor <br/>RANGE: [Person](/schemas#person) |
| data.object | object | reference to the item on which the offer is being made  |
| data.instrument | * | thing used by the agent to perform the action, i.e. an application or website that generated the message.  |
| data.id | string&lt;uri&gt; | Linked-Data uri  |
| data.name | string | the name of the item.  |
| data.url | string&lt;uri&gt; | a public href that is clearly and uniquely associated with the item.  |
| data.recipient | object | the offer recipient <br/>RANGE: [Person](/schemas#person),[Organization](/schemas#organization) |
| data.offerPrice | object | a price offered for transfer of ownership of an item <br/>RANGE: [PriceSpecification](/schemas#pricespecification) |
| source | string&lt;uri&gt;! | the event origin, typically a user  |
| instrument | string&lt;uri&gt; | event producer, typically an app or service  |
