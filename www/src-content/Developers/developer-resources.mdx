## Libraries & Tools

- [Yodata Context](https://www.npmjs.com/package/yodata-context): simple, robust JSON data transformations without (much) code. 
- [Yodata Event Router](https://www.npmjs.com/package/@yodata/event-router): create and deploy event handling micro-services
- [Yodata Actions](https://github.com/Yodata/yodata-actions): produce consistent, compliant structured data in your applications
- [Yodata Inbox Poller](https://www.npmjs.com/package/yodata-inbox-poller): inbox polling service scaffold, easy deploy with docker

## Specifications
- [Schema.org](https://schema.org) - mother of all all rdf schema vocabularies
- [Resource Description Framework (RDF)](https://www.w3.org/standards/techs/rdf#w3c_all) - universal data serialization standard
- [Linked Data Notifications](https://www.w3.org/TR/ldn/) - decentralized pub/sub
- [SOLID - Social Linked Data Application Framework](https://github.com/solid)
