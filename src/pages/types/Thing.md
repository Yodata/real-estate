## Example



```json
{
  "type": "string",
  "@id": "http://example.com"
}
```

## Schema

| Name | Type | Description |
|---|---|---|
| (root) | object | - |
| type | string | The item type (Linked-Data @type) pattern (`^[A-Z][a-zA-Z0-9]+$`) |
| @id | string | the liked data uri for the Thing format (`uri`) |
| (property names) | - |  pattern (`^[a-z@$][a-zA-Z0-9-_]+$`) |
