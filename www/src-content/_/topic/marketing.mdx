# marketing

Marketing events


### marketing Events

| Event | Description |
| :---- | :---------- |
| [realestate/marketingprogram#create](#create) | a marketing program has been created |

## create

a marketing program has been created


```json
{
  "topic": "realestate/marketingprogram#create",
  "data": {
    "type": "CreateAction",
    "object": {
      "type": "MarketingProgram",
      "name": "Market Activity Report 508 Homewood Ave",
      "member": [
        {
          "type": "ProgramMembership",
          "roleName": "Owner",
          "memberOf": null,
          "member": {
            "type": "Person",
            "id": "http://{user}.example.com/profile/card#me"
          },
          "startDate": "2019-05-22T02:19:48Z",
          "endDate": "2019-05-22T02:19:48Z"
        }
      ],
      "creator": "http://{user}.example.com/profile/card#me",
      "dateCreated": "2019-05-22T02:19:48Z",
      "dateModified": "2019-05-22T02:19:48Z"
    },
    "agent": "http://{agent}.example.com/profile/card#me"
  },
  "source": "http://{user}.example.com/profile/card#me",
  "instrument": "http://{supercrm}.example.com/profile/card#me",
  "time": "2019-05-22T02:19:48Z"
}
```


| Name | Type | Description |
|:-----| :--- | :---------- |
| topic | string! | {domain}/{topic}#{event}  |
| data | object! | the object was created by the agent  |
| data.type | string! | CreateAction  |
| data.object | object | The object upon which the action is carried out  |
| data.agent | string | the direct performer of the action  |
| source | string&lt;uri&gt;! | the event origin, typically a user  |
| instrument | string&lt;uri&gt; | event producer, typically an app or service  |
| time | string&lt;date-time&gt; | when the event was published  |
