# Schemas

## Action

The base model for events, based on https://schema.org/Action

| Name/Type | Type | Description |
|:--------- | :--- | :---------- |
| type | string&lt;regex&gt;! | override the type property to ensure Action is used  |
| actionStatus | string | Indicates the current disposition of the Action.  |
| agent |  | The direct performer or driver of the action (animate or inanimate). e.g. John wrote a book. <br/>RANGE: [Organization](/schemas#organization),[Person](/schemas#person) |
| agent.1 | [Person](/schemas#person) | Person  |
| endTime | string&lt;date-time&gt; | The endTime of something. For a reserved event or service (e.g. FoodEstablishmentReservation), the time that it is expected to end. For actions that span a period of time, when the action was performed. e.g. John wrote a book from January to December.  |
| error | object | For failed actions, more information on the cause of the failure.  |
| instrument | object | The object that helped the agent perform the action. e.g. John wrote a book with a pen.  |
| location |  | The location of for example where the event is happening, an organization is located, or where an action takes place.  |
| location.1 | object! | PostalAddress  |
| location.2 | string |   |
| object | object | The object upon the action is carried out, whose state is kept intact or changed. Also known as the semantic roles patient, affected or undergoer (which change their state) or theme (which doesn't). e.g. John read a book.  |
| participant |  | Other co-agents that participated in the action indirectly. e.g. John wrote a book with Steve.  |
| participant.1 | object! | Person  |
| participant.2 | [undefined] |   |
| participant.2.1 | object! | Person  |
| result | object | The result produced in the action. e.g. John wrote a book.  |
| startTime | string&lt;date-time&gt; | The startTime of something. For a reserved event or service (e.g. FoodEstablishmentReservation), the time that it is expected to start. For actions that span a period of time, when the action was performed. e.g. John wrote a book from January to December.  |
| target | object | Indicates a target EntryPoint for an Action.  |

```json
{
  "type": "string",
  "actionStatus": "ActiveActionStatus",
  "agent": {
    "type": "Organization",
    "telephone": "079.706.7065 x6341",
    "email": "user@example.com",
    "faxNumber": "(873) 271-4802",
    "contactPoint": [
      {
        "type": "ContactPoint",
        "name": "Work",
        "telephone": "1-364-127-9618 x20418",
        "faxNumber": "834.320.1602",
        "email": "user@example.com"
      }
    ],
    "parentOrganization": [
      "http://example.com"
    ]
  },
  "endTime": "2019-02-18T20:28:43Z",
  "error": {
    "type": "string"
  },
  "instrument": {
    "type": "string"
  },
  "location": {
    "type": "string"
  },
  "object": {
    "type": "string"
  },
  "participant": {
    "type": "string"
  },
  "result": {
    "type": "string"
  },
  "startTime": "2019-02-18T20:28:43Z",
  "target": {}
}
```
