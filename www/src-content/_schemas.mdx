# Object Types



## Action




| Name/Type | Type | Description |
|:--------- | :--- | :---------- |
| topic | [string] | the subscription type(s) assigned to the action  |
| type | string! |   |
| actionStatus | string | Indicates the current disposition of the Action.  |
| agent | string&lt;uri&gt;&#124;object | the direct performer or driver of the action. <br/>RANGE: [Person](/schemas#person),[Organization](/schemas#organization),[URI](/schemas#uri) |
| instrument | string&lt;uri&gt; | thing used by the agent to perform the action, i.e. an application or website that generated the message.  |
| object | object&#124;string&lt;uri&gt; | The object upon the action is carried out, whose state is kept intact or changed.  |
| participant | [object] | Other co-agents that participated in the action indirectly. <br/>RANGE: [Person](/schemas#person),[Organization](/schemas#organization) |
| location | object | The location of for example where the event is happening, an organization is located, or where an action takes place. <br/>RANGE: [Place](/schemas#place),[PostalAddress](/schemas#postaladdress),[Text](/schemas#text) |
| result | object | a result produced in the action. <br/>RANGE: [Thing](/schemas#thing) |
| startTime | string&lt;date-time&gt; | the date & time when something begins  |
| endTime | string&lt;date-time&gt; | The endTime of something.  |
| target | object | Indicates a target EntryPoint for an Action. <br/>RANGE: [EntryPoint](/schemas#entrypoint) |
| error | object | For failed actions, more information on the cause of the failure.  |

```json
{
  "topic": [
    "profile:read"
  ],
  "type": "string",
  "actionStatus": "CompletedActionStatus",
  "agent": "http://user.example.com/profile/card#me",
  "instrument": "http://app.example.com/profile/card#me",
  "object": {
    "type": "Thing",
    "id": "http://example.com/linked/data/uri"
  },
  "participant": [
    {}
  ],
  "location": {},
  "result": {},
  "startTime": "2019-02-21T21:19:48Z",
  "endTime": "2019-02-21T21:19:48Z",
  "target": {},
  "error": {}
}
```
