## add

a contact was added/created.


* foo
* bar


```json
{
  "topic": "realestate/contact#add",
  "data": {
    "type": "AddAction",
    "object": {
      "type": "Contact",
      "name": "Bruce Wayne",
      "email": "batman@example.com"
    },
    "agent": null,
    "instrument": "http://example.com",
    "targetCollection": {
      "type": "Collection",
      "id": "https://{user}.example.com/data/contact/"
    }
  },
  "source": "http://{user}.example.com/profile/card#me",
  "instrument": "http://{supercrm}.example.com/profile/card#me",
  "time": "2019-05-22T12:52:29Z"
}
```


| Name | Type | Description |
|:-----| :--- | :---------- |
| topic | string! | {domain}/{topic}#{event}  |
| data | object! |   |
| data.type | string! | the action type  |
| data.object | object | a contact  |
| data.agent | * | the direct performer of the action  |
| data.instrument | string&lt;uri&gt; | something used by the agent to perform the action, i.e. an application or website that generated the message.  |
| data.targetCollection | * | the collection receiving the added object  |
| source | string&lt;uri&gt;! | the event origin, typically a user  |
| instrument | string&lt;uri&gt; | event producer, typically an app or service  |
| time | string&lt;date-time&gt; | when the event was published  |


## update

```json

  "topic": "realestate/contact#update",
  "data": {
    "type": "UpdateAction",

      "type": "Contact",
      "name": "the updated contact"
    },
    "agent": null,
    "instrument": "http://example.com",
    "targetCollection": {
      "type": "Collection",
      "name": "targetcollection"
    }
  },
  "source": "http://{user}.example.com/profile/card#me",

  "time": "2019-05-22T12:52:29Z"
}
```


| Name | Type | Description |
