# marketing

Marketing events


### marketing Events

| Event | Description |
| :---- | :---------- |
| [realestate/marketingprogram#create](#create) | a marketing program has been created |
| [realestate/marketingprogram#memberadd](#memberadd) | a program member was added by the agent |
| [realestate/marketingprogram#memberremove](#memberremove) | a program member was removed by the agent |


## create

a marketing program has been created


```json
{
  "topic": "realestate/marketingprogram#create",
  "data": {
    "type": "CreateAction",
    "object": {
      "type": "MarketingProgram",
      "name": "Market Activity Report 508 Homewood Ave",
      "about": {
        "type": "Place",
        "address": {
          "streetAddress": "508 Homewood Ave",
          "addressLocality": "Chula Vista",
          "addressRegion": "California",
          "postalCode": "55555"
        }
      }
    },
    "agent": "http://{agent}.example.com/profile/card#me",
    "instrument": "https://{app}.example.com/profile/card#me"
  },
  "source": "http://{user}.example.com/profile/card#me",
  "instrument": "http://{supercrm}.example.com/profile/card#me",
  "time": "2019-05-22T14:53:14Z"
}
```


| Name | Type | Description |
|:-----| :--- | :---------- |
| topic | string! | the message id, equivalent to cloudevents type.  |
| data | object! | the object was created by the agent  |
| data.type | string! | CreateAction  |
| data.object | object | the object upon which the action is carried out <br/>RANGE: [MarketingProgram](/schemas#marketingprogram) |
| data.agent | string | the direct performer of the action  |
| data.instrument | string&lt;uri&gt; | something used by the agent to perform the action, i.e. an application or website that generated the message.  |
| source | string&lt;uri&gt;! | the subscription source  |
| instrument | string&lt;uri&gt; | the message producer  |
| time | string&lt;date-time&gt; | the time the message was produced  |


## memberadd

a program member was added by the agent


```json
{
  "topic": "realestate/marketingprogram#memberadd",
  "data": {
    "type": "AddAction",
    "object": {
      "type": "ProgramMembership",
      "role": "Owner",
      "member": {
        "type": "Person",
        "name": "Edgardo Fisher",
        "identifier": {
          "aceid": "816fd12f-6479-4f45-b26f-0a2ba54972b6"
        }
      },
      "memberOf": {
        "type": "MarketingProgram",
        "name": "Market Activity Report 508 Homewood Ave",
        "identifier": {
          "aceid": "4ef1469a-1fc4-43af-96b1-0e4402ccc46e",
          "buysideid": "83271c2a-c35e-44e9-b790-5389f2c11b17"
        }
      }
    },
    "agent": "https://{user}.example.com/profile/card#me",
    "instrument": "https://{app}.example.com/profile/card#me",
    "targetCollection": {
      "type": "MarketingProgram",
      "id": "https://{user}.example.com/data/contact/",
      "name": "Market Activity Report 508 Homewood Ave",
      "identifier": {
        "aceid": "4ef1469a-1fc4-43af-96b1-0e4402ccc46e",
        "buysideid": "83271c2a-c35e-44e9-b790-5389f2c11b17"
      }
    }
  },
  "source": "http://{user}.example.com/profile/card#me",
  "instrument": "http://{supercrm}.example.com/profile/card#me",
  "time": "2019-05-22T14:53:14Z"
}
```


| Name | Type | Description |
|:-----| :--- | :---------- |
| topic | string! | the message id, equivalent to cloudevents type.  |
| data | object! | the message payload  |
| data.type | string! | AddAction  |
| data.object | object | describes membership relation between a member (Person) and a MarketingProgram <br/>RANGE: [ProgramMembership](/schemas#programmembership) |
| data.agent | * | the direct performer of the action  |
| data.instrument | string&lt;uri&gt; | something used by the agent to perform the action, i.e. an application or website that generated the message.  |
| data.targetCollection | object | the collection receiving the added object <br/>RANGE: [MarketingProgram](/schemas#marketingprogram) |
| source | string&lt;uri&gt;! | the subscription source  |
| instrument | string&lt;uri&gt; | the message producer  |
| time | string&lt;date-time&gt; | the time the message was produced  |
