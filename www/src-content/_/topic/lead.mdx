## accept

a lead (object) was accepted by the agent


```json
{
  "topic": "realestate/lead#accept",
  "data": {
    "type": "AcceptAction",
    "object": {
      "type": "Lead",
      "object": {
        "type": "Person",
        "name": "Larry Lead",
        "email": "larry@example.com"
      },
      "recipient": "http://{user}.example.com/profile/card#me",
      "sender": "http://{provider}.example.com/profile/card#me",
      "dateSent": "2019-05-22T13:21:33Z",
      "activity": [
        {
          "type": "Action",
          "object": {},
          "agent": null,
          "instrument": "https://{app}.example.com/profile/card#me"
        }
      ]
    },
    "agent": null,
    "instrument": "https://{app}.example.com/profile/card#me"
  },
  "source": "http://{user}.example.com/profile/card#me",
  "instrument": "http://{supercrm}.example.com/profile/card#me",
  "time": "2019-05-22T13:21:33Z"
}
```


| Name | Type | Description |
|:-----| :--- | :---------- |
| topic | string! | realestate/lead#accept  |
| data | object! | The act of committing to/adopting an object.  |
| data.type | string! | the action type  |
| data.object | [Lead](/schemas#lead) | a lead offered or assigned by the agent to one or more recipients  |
| data.agent | * | the person or organization accepting the item  |
| data.instrument | string&lt;uri&gt; | something used by the agent to perform the action, i.e. an application or website that generated the message.  |
| source | string&lt;uri&gt;! | the subscription source  |
| instrument | string&lt;uri&gt; | the message producer  |
| time | string&lt;date-time&gt; | the time the message was produced  |

  "instrument": "http://{supercrm}.example.com/profile/card#me",

| Name | Type | Description |
