# Objects



## AppointmentRequest
```json
{
  "type": "AppointmentRequest",
  "agent": null,
  "object": {
    "type": "Comment",
    "text": "Winter is coming",
    "author": {
      "type": "Person",
      "email": "user@example.com"
    },
    "about": {
      "type": "RealEstateListing",
      "id": "http://example.com/listing/1"
    }
  },
  "instrument": "https://{app}.example.com/profile/card#me",
  "actionStatus": "CompletedActionStatus"
}
```


| Name | Type | Description |
|:-----| :--- | :---------- |
| type | string! | the action type  |
| agent | * | the direct performer of the action  |
| object | object | A comment on an item.  |
| object.type | string! | Comment  |
| object.id | string&lt;uri&gt; | Linked-Data uri  |
| object.name | string | the name of the item.  |
| object.url | string&lt;uri&gt; | a public href that is clearly and uniquely associated with the item.  |
| object.author | [Person](/schemas#person) |   |
| object.text | string | the comment content  |
| object.about | object | the subject of the item  |
| instrument | * | thing used by the agent to perform the action, i.e. an application or website that generated the message.  |
| actionStatus | string | disposition of the Action when the event was created  |
