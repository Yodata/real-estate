# contact

Contact events


### contact Events

| Event | Description |
| :---- | :---------- |
| [realestate/contact#add](#add) | a contact was added/created. |

## add

a contact was added/created.


```json
{
  "topic": "realestate/contact#add",
  "data": {
    "type": "AddAction",
    "object": null
  },
  "source": "http://{user}.example.com/profile/card#me",
  "instrument": "http://{supercrm}.example.com/profile/card#me",
  "time": "2019-05-22T02:19:48Z"
}
```


| Name | Type | Description |
|:-----| :--- | :---------- |
| topic | string! | {domain}/{topic}#{event}  |
| data | object! |   |
| data.type | string |   |
| data.object | [Contact](/schemas#contact) |  <br/>RANGE: [Contact](/schemas#contact) |
| source | string&lt;uri&gt;! | the event origin, typically a user  |
| instrument | string&lt;uri&gt; | event producer, typically an app or service  |
| time | string&lt;date-time&gt; | when the event was published  |


## update

a crm contact was updated


```json
{
  "topic": "realestate/contact#update",
  "data": {
    "type": "UpdateAction",
    "object": {
      "type": "Contact",
      "name": "the updated contact"
    },
    "agent": {
      "type": "Person",
      "id": "http://{user}.example.com/profile/card#me"
    },
    "targetCollection": {
      "type": "Collection",
      "name": "targetcollection"
    }
  },
  "source": "http://{user}.example.com/profile/card#me",
  "instrument": "http://{supercrm}.example.com/profile/card#me",
  "time": "2019-05-22T02:19:48Z"
}
```


| Name | Type | Description |
|:-----| :--- | :---------- |
| topic | string! | {domain}/{topic}#{event}  |
| data | object! |   |
| data.type | string! | UpdateAction  |
| data.object | object | The object upon which the action is carried out  |
| data.agent | * | the direct performer of the action  |
| data.targetCollection | object | target collection  |
| source | string&lt;uri&gt;! | the event origin, typically a user  |
| instrument | string&lt;uri&gt; | event producer, typically an app or service  |
| time | string&lt;date-time&gt; | when the event was published  |
